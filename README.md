🟦 2. 기능 소개
### 2.1组队 / 채팅그룹 기능
✔ 러닝(릴레이)

형태: 릴레이 중심

구성:

전반부: 본인이 주러너, 이전 사람은 동반러너

후반부: 본인은 동반러너, 다음 사람이 주러너

릴레이 방식:

이전 주자가 중간 지점 도착 → “릴레이 버튼” → 다음 주자 타이머 시작

두 사람이 함께 달리며, 다음 주자가 릴레이 버튼을 누르면 이전 주자는 완료 처리

✔ 팀장 권한

다음 주자에게 영향 없도록 특정 사람 건너뛰기 가능

✔ 필터 & 검색

홈: 전체 / 지역 / 시간대 / 강도 필터

팀 신청 메뉴: 카드 리스트로 팀 표시

지역(구) 단위 기본 표시, 시간대 2시간 단위 설정

✔ 팀 카드 정보

구 이름

출발/도착 지점

시작·종료 시간

팀 인원 수

강도(intensity level = 평균 속도)

“팀 가입 신청” 버튼 → 팀장 승인 후 가입

✔ 신청서

건강 상태

이름/나이/성별(자동)

직업

신청 이유

희망 구간

✔ 구간 제한

구간당 최대 2명

전체 구간 채워져야 러닝 시작 가능

릴레이 흐름: 팀장 시작 → 중간 지점에서 릴레이 → 다음 사람에게 이동… 마지막 사람까지 반복

✔ 승인/거절

승인 → 팀 참여

거절 → 사유 알림

✔ 팀장 신청

코스 90% 이상 완주(신뢰도) 필요

직접 코스/강도/시간 설계
