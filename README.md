# 배경 및 기능 설계 일람

| 배경 | 설명 | 서비스 설계 | 기능 |
|------|------|-------------|-------|
| 치료 자원 접근성 부족 | 1. 치료가 가장 필요한 사람들이 오히려 치료에 접근하기 가장 어려움. 저소득층은 긴 대기 시간·높은 심리적 비용·약물 부작용을 감당하기 어려움. 청소년 등 취약 계층은 가족/집단 문제의 대표 환자로 취급되어 자율적 변화 어려움. <br> 2. 운동은 저비용·고효과의 항우울 개입이며 연구 근거가 탄탄함. | 효과적·저렴·접근성 높은 방안 | 러닝 선택 |
| 우울증의 일상 침투 | 우울은 행동력을 약화시키고 최소 생존만 유지하게 만듦. 지속적 실패 경험은 학습된 무기력을 심화시킴. 그러나 조기 각성, 부정적 인지 등의 ‘개입 기회’로 전환 가능. 최소 심리 비용으로 시작할 여지가 있음. | 우울 특성을 고려해 심리 비용 최소화 | 릴레이 러닝(이전 주자가 다음 주자의 시작 여부 결정 → 중도 포기 예방), 실시간 완주율 업데이트 |
| 과도한 의료화·낙인 문제 | 1. 현대 사회는 ‘정상’ 기준이 단일화되어 있어 환자가 ‘왜 이렇게 되었는가’만 주목됨. 정체성·욕구·능력은 무시됨. <br> • 낙인으로 인해 도움 요청 회피 <br> • 원래 의료가 필요 없는 사람까지 시스템으로 유입 <br> • 사회적 상호작용 자체가 부담이 되는 환자 존재 <br> 2. 의료인은 사회 기준의 수혜자이기 때문에 환자 일상을 깊이 이해하지 못하는 경우 다수. 인간은 매우 복잡하며 단순 의료만으로는 동기 활성화 어렵다. | 비슷한 경험을 가진 사람들끼리 연결하여 부담 감소 | 대상: 심각·반복적 우울 경험자 |
|  |  | 부정적 감정·행동을 받아들일 수 있는 공간 제공 | 익명 트리홀·DM·그룹 채팅 |
|  |  | 비사회적 부담이 적은 자연스러운 형태의 사회 활동 제공 | 릴레이 러닝: 전반부 주러너 + 후반부 동반러 구조, 생활 공유·감정 발산 기능 포함 |
|  |  | 인지 부담 최소화, 선택권 확대 | 과도한 가치관·이념 강요 없음, 강제적 사회 interaction 없음 |


🟦 2. 기능 소개
### 2.1组队 / 채팅그룹 기능
✔ 러닝(릴레이)

형태: 릴레이 중심

구성:

전반부: 본인이 주러너, 이전 사람은 동반러너

후반부: 본인은 동반러너, 다음 사람이 주러너

릴레이 방식:

이전 주자가 중간 지점 도착 → “릴레이 버튼” → 다음 주자 타이머 시작

두 사람이 함께 달리며, 다음 주자가 릴레이 버튼을 누르면 이전 주자는 완료 처리

✔ 팀장 권한

다음 주자에게 영향 없도록 특정 사람 건너뛰기 가능

✔ 필터 & 검색

홈: 전체 / 지역 / 시간대 / 강도 필터

팀 신청 메뉴: 카드 리스트로 팀 표시

지역(구) 단위 기본 표시, 시간대 2시간 단위 설정
